 一个制作平移图表类视频的工具  
# 功能
帮助创作对比类的平移视频，如：https://www.bilibili.com/video/BV1Ma411j71v  
目前的主要功能有:  
>1.将/images下的图片进行拼接  
>2.从左向右扫描图片生成预设帧率与时间长度的视频（可以想象一个摄像头取景框在从左向右的工作）。

本项目基于库PIL与MoviePY，支持的图片格式、视频输出格式、视频编码器和前述两个库一致，支持各类主流格式。
