目前的主要功能有:  
>1.将/images下的图片进行拼接  
>2.根据设置生成一个摄像头取景框，从左向右扫描图片生成预设帧率与时间长度的视频。

因为基于库PIL与MoviePY，所以本程序支持的图片格式、视频输出格式、视频编码器和前述两个库一致，支持各类主流格式。
